# GEMS-GER Starting Kit

This kit contains everything you need to participate in the challenge.

### Quick Start:
1. Install dependencies: `pip install -r requirements.txt`
2. Open `template_starting_kit.ipynb` in Jupyter or VS Code.
3. The data is already located in the `dev_phase/` folder.

### Submitting:
When ready, run `python zip_submission.py` to create your upload package.
